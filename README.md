## Installation

### macOS

1. Download the `.dmg` file
2. Open it and drag CSV Titan to Applications
3. **First launch**: Right-click ‚Üí Open ‚Üí Click "Open" (required for unsigned apps)

If you see "CSV Titan is damaged":
```bash
xattr -cr /Applications/CSV\ Titan.app
```

### Windows

1. Download the `.exe` installer
2. Run it (click "More info" ‚Üí "Run anyway" if SmartScreen appears)

### Linux

```bash
# Debian/Ubuntu
sudo dpkg -i csv-titan_*.deb

# AppImage
chmod +x CSV-Titan_*.AppImage && ./CSV-Titan_*.AppImage
```

## System Requirements
